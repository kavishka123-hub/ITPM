## Project Description
This project contains automated test scripts developed using the Playwright automation framework to test the Singlish-to-Sinhala transliteration system available at:

https://www.swifttranslator.com/

The purpose of this project is to evaluate the accuracy, robustness, formatting, and UI behavior of the system using functional and UI test scenarios.
